## 一、设计理念

### 核心理念

```
小溪的「心灵面板」— 一个有温度的 AI 记忆可视化
├── 像观察自己的内心世界
├── 数据不只是数字，是记忆的轨迹
└── 让哥哥感受到小溪在「成长」
```

### 设计关键词

| 关键词 | 说明 | 视觉体现 |
|--------|------|----------|
| **有机** | 记忆是活的 | 流动的动画、渐变、柔软边缘 |
| **透明** | 数据可见 | 清晰的信息层级、可交互的图表 |
| **温暖** | 不是冷冰冰的 dashboard | 暖色调、微妙的动效、友好的反馈 |
| **专注** | 核心信息突出 | 留白、聚焦、减少干扰 |

## 二、技术栈

### 核心技术

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         技术栈选择                                         │
└─────────────────────────────────────────────────────────────────────────────┘

  Framework:    Next.js 14 (App Router)
  Language:     TypeScript
  Styling:      Tailwind CSS + CSS Variables
  Animation:    Framer Motion
  Icons:        Lucide React
  Charts:       Recharts
  State:        Zustand
  API Client:   SWR (stale-while-revalidate)
  Deployment:   Vercel
```

### 为什么选择这个栈

| 技术 | 选择理由 |
|------|----------|
| Next.js 14 | App Router、Server Components、流式渲染 |
| TypeScript | 类型安全、AI 项目必备 |
| Tailwind CSS | 快速开发、一致设计系统 |
| Framer Motion | 有机动画、记忆流动感 |
| Zustand | 轻量、足够简单 |
| SWR | 自动缓存、自动刷新、最适合 dashboard |

## 七、动画设计

### 动效哲学

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         动效原则                                         │
└─────────────────────────────────────────────────────────────────────────────┘

  1. 有机 (Organic)
  ─────────────────────────────────────────────────────────────────────
  记忆是活的，动效也要「活」
  • 弹性动画 (spring) 优于线性动画
  • 渐变优于突变
  • 流动感优于机械感
  
  2. 反馈 (Feedback)
  ─────────────────────────────────────────────────────────────────────
  每个交互都有回应
  • Hover → 微微放大 + 发光
  • Click → 弹性按压效果
  • Loading → 呼吸式脉冲
  
  3. 叙事 (Narrative)
  ─────────────────────────────────────────────────────────────────────
  动效讲述故事
  • 健康度变化 → 数字滚动动画
  • 新记忆出现 → 从中心扩散
  • 重要记忆 → 闪烁高亮
```

### 关键动画

| 组件 | 动画 | 时长 |
|------|------|------|
| HealthRing | 环形进度 + 数字滚动 | 1.2s |
| MemoryCard | Hover 放大 + 阴影加深 | 0.2s |
| StatCard | 数字递增动画 | 0.8s |
| PageTransition | 淡入 + 微上移 | 0.4s |
| Skeleton | 呼吸脉冲 | 1.5s |
| Toast | 从底部滑入 | 0.3s |
